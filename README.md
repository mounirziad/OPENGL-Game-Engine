Modern C++ Game Engine
A professional-grade 3D game engine built entirely from scratch in modern C++ with OpenGL. This engine showcases advanced graphics programming techniques, custom Entity-Component-System architecture, and real-time rendering capabilities.

 Features
Core Architecture
Custom Entity-Component-System (ECS) - Type-safe component storage and efficient entity management

Memory-Optimized Design - Real-time memory tracking and efficient resource management

Modular System Design - Clean separation of rendering, input, and game logic

Advanced Rendering
Physically-Based Rendering (PBR) - Realistic lighting with Cook-Torrance BRDF

Multi-Shader Support - Runtime switching between Phong, PBR, Wireframe, and Flat shading

Dynamic Material System - Real-time material properties and shader management

Deferred Rendering Pipeline - Optimized for complex scenes with multiple light sources

 Engine Systems
Asset Management - Runtime model loading with support for OBJ formats

Input System - Flexible camera controls and input handling

Real-time Debug UI - Integrated Dear ImGui for development tools

Camera System - Free-fly camera with smooth movement and mouse look
